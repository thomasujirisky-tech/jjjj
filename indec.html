Cara Upload agar Langsung Aktif di GitHub Pages

Cara Termudah (Tanpa Coding/Local Setup):

  1. Buat repository baru di akun GitHub Anda (misal beri nama sejarah-jepang).
  2. Klik tombol "uploading an existing file" pada halaman repository baru tersebut.
  3. Unggah file index.html ini ke dalam repository tersebut.
  4. Masuk ke tab Settings di bagian atas repository GitHub Anda.
  5. Di menu sebelah kiri, cari bagian Pages.
  6. Di bawah bagian Build and deployment, ganti Source ke Deploy from a branch.
  7. Atur Branch menjadi main (atau master) dan folders ke / (root), lalu klik Save.
  8. Tunggu sekitar 1-2 menit, lalu segarkan halaman. Tautan website Anda yang aktif akan muncul di bagian atas halaman Settings tersebut!
